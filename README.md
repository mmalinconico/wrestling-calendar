# Wrestling Calendar

An automatically updating iCalendar (.ics) subscription for major professional wrestling events.

## Included Promotions

- WWE Premium Live Events (PLEs)
- Saturday Night's Main Event
- Sunday Night's Main Event
- NXT Premium Live Events
- AEW Pay-Per-Views
- Ring of Honor (ROH) Pay-Per-Views

Weekly television shows and special TV episodes are intentionally excluded to keep the calendar focused on major events.

## Features

- Automatically scrapes upcoming event information from Wikipedia.
- Updates every 4 hours using GitHub Actions.
- Generates a standards-compliant iCalendar (.ics) feed.
- Hosted with GitHub Pages.
- Works with Apple Calendar, Google Calendar, Outlook, and other calendar apps that support subscriptions.
