# Wrestling Calendar

An automatically updating iCalendar (.ics) subscription for major professional wrestling events.

Subscribe once and your calendar stays automatically up to date.

## Included Events

- WWE Premium Live Events (PLEs)
- WWE Saturday/Sunday Night's Main Events
- NXT Premium Live Events (PLEs)
- WWE/AAA Supercards
- AEW Pay-Per-Views (PPVs)
- ROH Pay-Per-Views (PPVs)

Weekly television shows and television specials are intentionally excluded to keep the calendar focused on major events.

## Features

- Automatically scrapes upcoming event information from Wikipedia.
- Updates every 4 hours using GitHub Actions.
- Generates a standards-compliant iCalendar (.ics) subscription.
- Hosted with GitHub Pages.
- Compatible with Apple Calendar, Google Calendar, Outlook, and other apps that support calendar subscriptions.
